Open Data (Sains) set

Open data (sains) set merupakan data-data yang telah diperoleh dari hasil survei/eksperimen/penelitian/analisis yang diunggah terbuka oleh institusi agar dapat dimanfaatkan oleh periset/akademisi/publik. Pustaka Data dari Universitas Gadjah Mada merupakan salah satu inisiatif platform data set yang digunakan untuk menyimpan dan berbagi data-data penelitian/publik yang dipromosikan bagi komunitas IDREN.
